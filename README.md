## Функции

-  управлять списками просмотров
-  читать новости из мира аниме
-  просматривать календарь выхода новых серий
-  находить информацию о тайтлах
-  смотреть трейлеры, опенинги, эндинги

## Требования

- iOS 16.2+
- Xcode 14.2
